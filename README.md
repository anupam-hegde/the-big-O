# THE BIG O — Competitive Coding Club 🏆

A premium, interactive website for **THE BIG O** — a competitive coding club founded by 3rd-year students.

## ✨ Features

- 🌌 **3D Particle Constellation** — Three.js powered interactive hero section
- 🪐 **Holographic Team Cards** — 3D tilt, orbital avatar rings, holographic shimmer sweep
- 🎯 **Big O Notation Theme** — Clever coding-themed "Why Big O?" cards (O(1), O(log n), O(n), O(n!))
- ✨ **Custom Cursor** — Code-bracket rotating cursor with sparkle trail
- 🌀 **Scroll Animations** — Reveal effects, parallax, and smooth transitions
- 🎨 **Premium Glassmorphism** — Dark theme with neon accents and frosted glass effects

## 🚀 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/club-website.git
   ```

2. Open in browser:
   ```bash
   # Using Python
   python -m http.server 8000

   # Or simply open index.html in your browser
   ```

3. Visit `http://localhost:8000`

## 📁 Project Structure

```
├── index.html    # Main HTML page
├── styles.css    # All CSS styles
├── script.js     # JavaScript (Three.js, animations, interactions)

```

## 🛠️ Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Custom properties, keyframe animations, glassmorphism
- **JavaScript** — Vanilla JS with Three.js for 3D effects
- **Lucide Icons** — Icon library
